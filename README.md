# Тестовое задания Samedia. Часть 1

**Демонстрация:** https://shcherbina-andrei.github.io/Samedia-task1/

**Стек технологий:** HTML, CSS, Javascript

Реализована страница категорий. Использована методология БЭМ. Выполнен гибкий интерфейс, адаптирующийся под дисплеи различных устройств. Реализованы анимации: движения кругов (с использование key-frames) и изменение картинок при наведении на карточку категории, изменение положения листьев при движении мыши.
